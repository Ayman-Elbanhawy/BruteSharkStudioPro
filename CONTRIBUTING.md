# Contributing

First off, thanks for taking the time to contribute. BruteShark Studio welcomes contributions from everyone.

When contributing to this repository, please open or discuss the change through GitHub issues before making large behavioral changes.

## How Can You Contribute?
- Implementing new features from [BruteShark Studio Issues](https://github.com/Ayman-Elbanhawy/BruteSharkStudio/issues)
- Uploading example PCAP files, especially files, with interesting content.
- Proposing new features by [creating an issue](https://github.com/Ayman-Elbanhawy/BruteSharkStudio/issues)
- Reporting a bug by [creating an issue](https://github.com/Ayman-Elbanhawy/BruteSharkStudio/issues)
- Discussing the current state of the code
- Creating videos and example tutorials for BruteShark Studio
